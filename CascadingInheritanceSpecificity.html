Inheritance, Cascading and Specificity

Inheritance

Some properties and values are inheritable by elements from their parent/ancestor elements. This is true for most typography styles.

E.g. by changing the font family for the body element, all elements in the page ill inherit that font.

Cascading

The browser will always pick the last declaration in the document as long the selector is targeting the same element.
